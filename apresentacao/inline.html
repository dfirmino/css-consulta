O que é

É a forma como o elemento é exibido na tela.

Todo elemento por default pode ser inline ou block

Inline

Elementos inline são elementos que ocupam somente o tamanho necessário para sua exibição.

Ou seja, caso caiba 2 elementos inline na mesma linha, eles iram ocupar a mesma linha

ps: Não é possivel ajustar o tamanho de um elemento inline

Block

Elementos em bloco são elementos que mesmo que não sejam necessário para sua exibição uma linha inteira.

Ele irá sempre ocupar a linha inteira e jogar os próximos elementos para a linha de baixo.

Ps: é possivel ajustar o tamanho de elementos de bloco.

Inline-block

São elementos que se comportam como elementos inline, porém com caracteristicas de bloco.

Em outras palavras, embora os elementos ocupem a mesma linha é possível ajustar seu tamanho caso seja necessário
