O QUE É

CSS é uma linguagem de estilo que serve para definir a formatação de documentos HTML

Características

CSS = Cascade Style Sheets

Separa Formatação de conteúdo

W3C - Desenvolve/Mantém as especificações

CSS3 é a versão atual

Anatomia

}

Como Usar

Podemos utilizar dentro da tag STYLE do html (não recomendado)

Podemos criar um arquivo externo e "linkar" com a página atráves da tag link (recomendado)
