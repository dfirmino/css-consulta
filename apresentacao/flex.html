O QUE É

É uma nova forma de organizar layouts

Permite organizar os elementos de forma inteligente, não mais com tamanhos fixos

Terminologia

A partir do momento que definimos um elemento como flex ele se torna um Flex Container

Os elementos dentro de um flex container são chamados de Flex Items

Flex itens vão trabalhar baseados em sua flex Direction

Flex direction são baseados em eixos em que os flex items serão exibidos

O eixo principal que por padrão é na horizontal é chamdo de Main Axis

O eixo secundário que por padrão é na vertical é chamdo de Cross Axis

Conceitos

Container

Para definirmos um flex container basta adicionar o display como flex

Propriedades de Flex Container

Flex Direction

Define o a posição do main axis

Row: Horizontal

Row Reverse: Horizontal mas com os elementos invertidos

column: Vertical

column Reverse: Vertical mas com os elementos invertidos

Flex Wrap

Define se os flex itens vão ir para proxima linha caso não caibam no flex container

Justify Content

Controla o alinhamento de flex itens no Main Axis

Align Items

Controla o alinhamento de flex itens no Cross Axis

Align Content

Controla o alinhamento dos itens distribuidos em mais de uma linha no container

Propriedades de Flex Itens

Order

Define a ordem de exibição dos itens

Align Self

Define o alinhamento do item no Cross Axis

Flex Grow

Define se o item irá se comportar caso exista espaço sobrando no container

Calcula o espaço que está sobrando do container e divide esse espaço de acordo com o valor de cada item

Flex Shrink

Define se o item irá se comportar caso esteja faltando espaço no container

Calcula o espaço que está ultrapassando do container e remove esse espaço de acordo com o valor de cada item

Flex basis

Define o valor default do tamanho de um item em seu main axis dentro de um container

Esse comportamento pode ser alterado caso haja espaço sobrando no container e esteja com a propriedade flex grow aplicada

Flex

É um atalho para se trabalhar com as propriedade Flex Grow, basis e shrink

Flex: [flex-grow] [flex-shrink] [flex-basis]

Valor Padrão -> Flex: 0 1 auto
